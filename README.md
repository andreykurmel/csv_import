
## Installation

- clone project
- set up the .env file
- create {$DB_DATABASE} in MySql
- run commands:
> php artisan key:generate;

> composer install;

> npm install;

> php artisan migrate;

> php artisan db:seed;

> npm run prod

> php artisan queue:work --tries=3;

Also you can config Pusher for viewing the import progress.
